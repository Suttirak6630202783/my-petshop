PETSHOP Inventory System

เว็บแอปพลิเคชันสำหรับจัดการสินค้าในร้าน Petshop
พัฒนาโดยใช้ PHP + MySQL สำหรับฝึกการทำระบบ CRUD และ Authentication

Features
ระบบ Login / Register ผู้ใช้
แสดงรายการสินค้า (Product List)
เพิ่ม / แก้ไข / ลบสินค้า (CRUD)
อัปโหลดรูปภาพสินค้า
ดูรายละเอียดสินค้า (Detail Page)
ระบบจัดการ Session และ Logout

Tech Stack
Frontend:
HTML
CSS
Backend:
PHP (Native PHP)
Database:
MySQL
